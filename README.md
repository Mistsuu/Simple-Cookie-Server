<b>SET</b> ```SET <key0>=<value0>; <key1>=<value1>; <key2>=<value2>; (...)  # <- Ends with ;``` <br>
<b>GET</b> ```GET <key>;                                                    # <- Also ends with ; (JUST 1 KEY GET AT A TIME!!)```
